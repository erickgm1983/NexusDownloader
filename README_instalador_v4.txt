
📦 Nexus Downloader v4 – Manual de uso

Este programa te permite descargar videos de YouTube fácilmente, sin instalar nada adicional.

───────────────────────────────
🚀 ¿Cómo usarlo?
───────────────────────────────
1. Ejecuta NexusDownloader_v4.exe
2. Pega el enlace del video de YouTube.
3. Selecciona la calidad que deseas (SD, HD, 2K, 4K).
4. Presiona "Descargar Video".
5. El archivo se guardará como MP4 y se abrirá la carpeta automáticamente.

───────────────────────────────
